## [1.0.0] - 2026-07-09

### Added

#### Initial Public Release

This is the first open-source release of **PapsAI XNet**, accompanying the manuscript:

> **Explainable Hybrid Deep Learning for Automated Cervical Cytology Classification**

submitted to **Nature Scientific Reports**.

### Features

- Attention-guided CNN feature extraction.
- Channel Attention Module (CAM).
- Feature Normalization Layer.
- Adaptive Neuro-Fuzzy Inference System (ANFIS).
- Gaussian Membership Functions.
- Compact fuzzy rule base.
- Explainable AI framework for cervical cytology classification.
- Lightweight architecture suitable for deployment on resource-constrained platforms.
- Computational complexity analysis.
- Model evaluation framework.
- ROC analysis.
- Confusion matrix generation.
- Explainability visualizations.
- Reproducible training pipeline.

## Versioning Policy

PapsAI XNet follows Semantic Versioning.

- **Major versions** introduce significant architectural changes or new functionality.
- **Minor versions** introduce backward-compatible improvements.
- **Patch versions** include bug fixes, documentation updates, and maintenance improvements.
